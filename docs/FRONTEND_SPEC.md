
# HadassahHub Frontend Design Specification

## Product Vision
HadassahHub should feel like a modern EdTech platform for computer science students.

The design must be:
- clean
- modern
- minimal
- friendly
- easy to navigate

## Theme
Light theme with large white spaces and soft shadows.
